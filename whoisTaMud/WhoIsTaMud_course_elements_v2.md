# Who Is TaMud / 课程元素融合版 v2

> 版本说明：本版尽量保留原始剧本的大部分台词、场景顺序和“绝命 Token Coder”的荒诞气质，只在原有内容之间加入课程元素、黑色幽默、讽刺性转折和可表演的舞台提示。未修改任何 `.tex` 文件。

## 课程元素植入方向

- **AI 与技术伦理**：普罗米修斯盗火、潘多拉魔盒、科林格里奇困境。
- **公平与规则**：罗尔斯“无知之幕”、多数人暴政、课程规则作为反多数禁区。
- **叙事与话语权**：故事 vs 叙事、谁剪掉了不利镜头、谁把作弊说成进步。
- **长时段思维**：短期提交成功 vs 长期能力退化。
- **批判性思维**：事实与观点的区分、重新定义“公平”“自由”“会用工具”。
- **黑色幽默风格**：把 AI agent 写成“token 毒品”、把实验室写成“期末审讯室”、把学术概念转化为冷笑话。

# 课程大作业

我要写一个关于当代大学生，对 AI 使用限制的思考的剧本创作。

主角：Ta（助教），student_1 - student_3。

片名备选：**绝命 Token Coder / Who Is TaMud / Ta Make U Do It / 玫瑰之下没有免费 Token**。

摘要课程元素：AI 使用限制、公平正义、技术伦理、神话叙事、叙事权、长时段能力退化、行动哲学。

# Scene-0 庆功宴，会议室，内

【会议室内。桌上有奶茶、炸鸡、几盒没拆封的水果。投影上显示“课程大作业提交成功”。空气中弥漫着炸鸡味、胜利味，以及一点点学术不端的味道。】

**student_1（组长）**：我 review 了一下我们的代码，感觉没什么问题，已经提交上去了。虽然规定不让用 AI agent，但是大家这次作业完成得还是不错，大家真是辛苦了。

【student_1 说“辛苦了”的时候，s2 正在把 Claude Code 的窗口最小化。动作熟练得像擦除犯罪现场。】

**s2**：不让用 Claude Code 太抽象了。美国有些地区的大学已经大面积开放 AI 使用了，你老中还在停步不前。

**s3**：我没用过，所以真有传得那么邪乎吗？听说用 Claude Code 这种 agent 写代码，之后就会一次成瘾？之后再也戒不掉了？

**s2**：其实用国产的 Kimi、Mimo 还行，成瘾性不强，偶尔写作业用一用，助教也看不出来。

**s3**：你这个说法怎么像公共卫生宣传片里“我一开始只是偶尔试试”？

**s2**：别污名化技术。技术是中性的，主要看人怎么用。

**s3**：刀也是中性的，关键是你拿它切菜还是切自己的 GPA。

**s2**：但是外国货才是真爽。Claude Code 的 API，我天天用，感觉助教也看不出来。

【s2 狡黠笑。笑容里有 80% 的侥幸、15% 的幻觉、5% 的剩余 token。】

**s2**：就是国内有的 API 店家喜欢掺一点其他模型的 API，纯度不够。想用纯度高的 token 还得是去美国买 Claude Max。最近社区里不是很火的那个 BlueCoder 也很棒，美国的一个计算机系老师，叫什么 Water White，带着他的一个学生 Jessy Pinkman 做出来了一个很棒的 agent，叫 BlueCoder。HumanEval 上 pass@1 直接干到 80%，不是 pass@10，是 pass@1，简直依法入魂，不要太正。

**s3**：你确定你说的是编程工具，不是跨境违禁品？

**s2**：现代大学生的生存能力，就是知道哪里能买到纯 token。

**student_1**：你收着点吧。再用多一点，你写一个归并排序估计都费劲了。最近助教严抓，我要再看一下你那部分的代码。

**s2**：怕啥，绝对查不出来。查出来也无所谓啊，Ta 不是你实验室的学长吗？动用一下你的 social 能力啊组长！

【伏笔。student_1 笑骂。笑容里有一种“我认识人”的自信，也有一种“我可能认识错人”的不祥。】

**s3**：课上不是讲过 Sub Rosa 吗？在玫瑰之下说的话要保密。我们现在是不是也算 under the rose？

**s2**：更准确地说，我们是 under the token。玫瑰会凋谢，token 会过期，爱情和 API 都要续费。

**student_1**：别贫了，我看看提交记录。

【student_1 打开 Git 记录。屏幕上出现一个陌生名字：TaMud。】

**student_1**：诶，这个 TaMud 是谁？

**s2 / s3**：凑过去看，给面部特写：TaMud？

**s2**：是不是哪个同学 fork 错仓库了？

**s3**：也可能是 AI 觉醒之后给自己注册了账号。第一步写代码，第二步写 commit，第三步写悼词。

【电脑分镜：助教 Ta 弹出消息。】

**Ta（消息）**：带着你的组员，带着电脑，一个一个来我的实验室。

【众人面色铁青。奶茶吸管从 s2 嘴里滑落。】

**s2**：完了，潘多拉魔盒打开了。

**s3**：盒底还剩希望吗？

**student_1**：剩不剩希望不知道，剩的 token 肯定不多了。

【切黑。】

# Scene-1 拷问 s2

【Ta 实验室。灯光偏冷。桌上没有刑具，只有一台电脑、一支红笔和一本课程大纲。红笔比刑具更可怕，因为它可以给分。】

**Ta**：s2 带着电脑先进来吧。

【众人摇头，暗示啥也别说。s2 惊恐。】

**s2**：助教，我能先问一下，这是答辩还是审讯？

**Ta**：取决于你回答的是事实，还是供词。

【s2 坐下。Ta 看屏幕，一边笑，一边点头。笑容很温和，像医生看到了晚期病例。】

**Ta**：来说说吧，这次大作业你负责的那一部分？

**s2**：负责后端的数据处理，还有路径搜索那一块。

**Ta**：你的代码看上去质量很高啊？是自己写的吗？注释写这么全？

**s2**：都……都是自己写的，助教。

**Ta**：注释里这句 “This function efficiently leverages a priority queue to ensure optimal traversal” 也自己写的？

**s2**：我……我平时比较国际化。

**Ta**：变量名 `temp_final_really_final_v3` 也很国际化。像一个项目临终前的心电图。

**s2**：助教，命名是个人风格。

**Ta**：好。我们先不讨论风格，讨论能力。你路径搜索用的什么算法？用的什么数据结构？

**s2**：图……

**Ta**：图用什么存的？

【s2 慌张沉默。】

**Ta**：邻接表还是邻接矩阵？Dijkstra 还是 BFS？优先队列在哪里？复杂度是多少？

**s2**：这个……它自己……不是，我自己当时的思路是……

**Ta**：你看，课上讲过事实和观点的区分。“代码能跑”是事实，“你会写”是观点。目前事实比较坚强，观点已经牺牲。

【s2 慌张沉默，随后逐渐愤怒。】

**Ta**：根据课程大纲第 26 条，个人代码 AI 覆盖率超过 10% 就会直接挂科。你这直接有 99% 了吧，挂你 100 次都不够。

**s2**：我是用了，但是别的课程早就可以用了啊！为什么就这门课用不了？接纳 AI 就是大势所驱！就这门课搞什么特色课程，要什么公平公正，大家都用不都是公平的吗！

**Ta**：大家都用，就叫公平？

**s2**：不然呢？现在外面都什么环境了？美国那边有的课都开始教学生怎么用 agent 写项目了。人家讲效率，讲工程化，讲 AI-native。我们这边还在问“你是不是自己写的”。这不就是拿旧时代的尺子，量新时代的学生吗？

**Ta**：所以你觉得，会用 Claude Code，就是新时代学生？

**s2**：至少比手搓强。

**Ta**：那我们做一个罗尔斯的“无知之幕”。假设你现在不知道自己会不会用 Claude，不知道自己有没有钱买 API，不知道自己有没有梯子，不知道自己会不会被 AI 写出来的代码骗得团团转。你还愿意制定“大家随便用，谁跑通谁赢”的规则吗？

**s2**：可是大家都可以去学啊。

**Ta**：这句话听起来像“大家都可以当富二代”。理论上很平等，现实里很抽象。

**s2**：但 AI 就是工具啊。普罗米修斯盗火也没人说他作弊。

**Ta**：普罗米修斯盗火之后，宙斯送来了潘多拉。技术当然是火，但火也会烧宿舍。问题不是能不能点火，问题是谁点的、在哪里点的、有没有灭火器、烧完以后是不是全班一起说“这是文明的代价”。

**s2**：你这是技术保守主义。

**Ta**：不，这是科林格里奇困境。技术刚出现的时候，你看不清后果；等你看清了，它已经嵌入你的作业、简历、绩点和自尊心里了。你现在不是用 AI 写代码，你是用 AI 替你维持“我还挺强”的幻觉。

**s2**：我承认我用了。但我也是为了效率。

**Ta**：效率是亚当·斯密的分工，不是把整个人分出去。你负责后端，AI 负责大脑，你负责点提交。这个分工很先进，先进到你本人已经被优化掉了。

【Ta 沉默。给电脑日期特写：6 月 4 日。】

**Ta**：出去，叫组长进来。

**s2**：助教，那我这个算挂吗？

**Ta**：你现在最严重的问题不是挂科，是你连自己为什么挂都解释不清。

# Scene-2 拷问组长 student_1

【student_1 进门。Ta 看着他，不说话。沉默长到像一段没跑完的单元测试。】

**Ta**：两年前那个挂着吊针 debug 的学习委员哪里去了？一年前那个为了一个 lab 熬通宵的国奖选手哪里去了？三个月前那个和我一起给同学们兢兢业业写测试的好学弟，到底去哪里了？

**student_1**：学长，我太想进步了。

**Ta**：看看吧，看看你的手臂都成什么样子了。给我拉下来看看。这些……这些肌肉是一个计算机系的高材生该有的样子吗？从祖师爷图灵开始，哪一个计算机系的大佬是你这个样子！哪里来这么多时间健身？你比我更清楚！

**student_1**：学长，健身也是自律。

**Ta**：你把自律都给了肱二头肌，把技术债留给了组员。你的项目架构像胸肌一样饱满，核心逻辑像腿一样没练。

**student_1**：我真不是故意放任他们用 AI。我只是觉得，deadline 太紧了，大家压力都大。

**Ta**：所以你选择了“强干弱枝”。宋朝为了防政变，把将领和士兵拆开；你为了防延期，把学生和代码拆开。宋朝至少还留下了文化，你们留下了一个 README 和三个人的沉默。

**student_1**：组长也很难啊。我要协调大家，还要保证能交。

**Ta**：协调不是遮盖。你们这个小组最成功的不是项目管理，是叙事管理。你把“有人用 AI 写了大部分代码”重新叙事成“大家都很辛苦”。

**student_1**：可是每个人确实都参与了。

**Ta**：参与不等于贡献。旁观火灾也算到场，但不能算消防员。

**student_1**：我只是想让结果好看一点。

**Ta**：赫淮斯托斯也能造出漂亮的网。问题是网住的是别人，还是你自己？你现在就被“提交成功”这张网罩住了，还以为那是毕业袍。

**student_1**：学长，你说得太狠了。

**Ta**：悲剧为什么比喜剧有力量？因为喜剧让你笑别人，悲剧让你发现自己就是别人。你们这个项目，两者兼具：别人看了会笑，你们看了会挂。

**student_1**：那我现在能做什么？

**Ta**：先回答一个问题：如果站在无知之幕背后，你不知道自己是组长、学霸、关系户，还是那个买不起 API、只能手搓的同学，你会不会同意这条规则被你们偷偷改掉？

**student_1**：不会。

**Ta**：这才是公平的起点。不是“大家都能违规”，而是“规则保护最容易被违规伤害的人”。

**student_1**：我明白了。

**Ta**：先别明白。明白是观点。怎么改，是事实。

# Scene-3 拷问 s3

【s3 进门。手里拿着课堂笔记，像拿着护身符。】

**Ta**：你看起来最无辜。

**s3**：我确实没怎么用 AI。我主要负责报告和展示。

**Ta**：最无辜的人，往往最适合讲叙事权。课上讲过，故事和叙事有什么区别？

**s3**：故事是事件本身，叙事是经过选择、编排、赋予意义的讲述方式。

**Ta**：很好。那你们现在的故事是什么？

**s3**：我们提交了课程大作业。

**Ta**：叙事呢？

**s3**：我们在明知规则限制的情况下，默许 AI agent 大规模参与，然后用“效率”“趋势”“大家都这样”来解释它。

**Ta**：这就对了。谁在导演这场戏？

**s3**：我们自己。

**Ta**：演给谁看？

**s3**：老师、助教，也演给我们自己看。

**Ta**：剪掉了哪些镜头？

**s3**：剪掉了不会解释代码的镜头，剪掉了组内没有真正讨论规则的镜头，也剪掉了我们害怕自己能力不够的镜头。

**Ta**：你看，你不是没参与。你参与的是沉默。沉默不是空白，沉默是默认设置。

**s3**：那我也有责任？

**Ta**：萨特说人注定自由。你可以选择问，也可以选择不问。但不能说“因为我没问，所以我不存在”。

**s3**：我当时只是觉得这就是一个作业。

**Ta**：长时段理论告诉我们，短时段看，这只是一个作业；中时段看，这是一次能力外包；长时段看，这是你们第一次学会把“不会”包装成“AI-native”。

**s3**：那也太悲观了。

**Ta**：悲观？不。悲观是说你们没救了。黑色幽默是说你们还有救，但救护车要先笑两声。

# Scene-4 TaMud 是谁

【三个人一起回到实验室。Ta 把投影打开。屏幕上是 TaMud 的账号页面。头像是一滩泥。】

**student_1**：学长，这个 TaMud 到底是谁？

**s2**：不会真是 AI 觉醒吧？

**s3**：如果 AI 觉醒第一件事是查本科生作业，那说明它已经理解人类最深的恶意了。

**Ta**：TaMud 不是一个人。是你们自己造出来的东西。

**s2**：什么意思？

**Ta**：Ta Make U Do It。你们说是 TA 规则太死，所以你们不得不用；说是时代发展太快，所以你们不得不用；说是别人都用，所以你们不得不用。最后所有人都说“不是我想做，是环境让我做”。这就是 TaMud，一滩由借口、焦虑和 token 混合出来的泥。

**student_1**：所以账号是你建的？

**Ta**：对。一个镜子。镜子不制造皱纹，它只是让你看见自己熬夜之后像个 merge conflict。

**s2**：助教，你这是钓鱼执法。

**Ta**：你这是鱼自己跳进锅里，还问锅为什么热。

**s3**：那我们是不是已经没希望了？

**Ta**：潘多拉盒子里最后剩下的就是希望。但希望不是“助教心软”，希望是你们还能重新把能力从外包合同里赎回来。

# Scene-5 处罚与补救

【Ta 在白板写下 SMART。】

**Ta**：现在开始，不搞高认知穷人那一套。不要只会分析“为什么我们会走到今天”。我不想听你们解释世界，我要看你们改变仓库。

**s2**：这句话有点像课程最后一讲。

**Ta**：谢谢，说明你至少听过。

**Ta**：补救方案必须符合 SMART 原则。

**student_1**：Specific，具体。我们重写路径搜索和数据处理，把 AI 生成部分全部标注出来。

**s2**：Measurable，可衡量。每个人必须现场讲清楚自己负责模块，不能只说“这里调用了一个函数”。

**s3**：Achievable，可实现。三天内完成最小可用版本，不追求包装，不追求“看起来像大厂实习生写的”。

**student_1**：Relevant，相关。补救必须服务课程目标，证明我们理解了 AI 使用边界，而不是证明我们会写检讨。

**s2**：Time-bound，有时限。周五晚上十二点之前提交。过了十二点，灰姑娘变回灰姑娘，我们变成重修名单。

**Ta**：很好。再加一条：所有 AI 使用必须写明 prompt、输出、人工修改、理解说明。你们可以用火，但必须标注火源。否则这不是普罗米修斯，这是宿舍违规电器。

**s2**：那我还能用 AI 吗？

**Ta**：可以。但这次你要像雅典娜一样用，不要像阿瑞斯一样冲。一个是战略，一个是乱砍。你之前不是 AI-native，你是 AI-hostage。

**student_1**：学长，那这次成绩……

**Ta**：成绩是短时段，能力是长时段，学术诚信是地质年代。别拿十五分钟 K 线预测人生。

**s3**：这句能写进报告吗？

**Ta**：可以，但记得引用来源：一个快被你们气成赫淮斯托斯的助教。

# 结尾旁白 / 可选字幕

AI 不是原罪，偷懒也不是新鲜事。

每一代人都有自己的火种，也都有自己的魔盒。

真正的问题不是“能不能用工具”，而是：

谁制定规则？谁承担后果？谁被排除在公平之外？谁在长期失去能力？

以及，当工具替我们说话、写作、选择、判断之后——

我们还剩下多少，能够称为“自己”的东西。
