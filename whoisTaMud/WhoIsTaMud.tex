\section*{Who Is TaMud}


\subsection*{课程元素植入方向}

\begin{itemize}
\item AI 与技术伦理：普罗米修斯盗火、潘多拉魔盒、科林格里奇困境。
\item 公平与规则：罗尔斯“无知之幕”、多数人暴政、课程规则作为反多数禁区。
\item 叙事与话语权：故事 vs 叙事、谁剪掉了不利镜头、谁把作弊说成进步。
\item 长时段思维：短期提交成功 vs 长期能力退化。
\item 批判性思维：事实与观点的区分、重新定义“公平”“自由”“会用工具”。
\end{itemize}

\section*{课程大作业：剧本}

关于当代大学生，对 AI 使用限制的思考的剧本创作。\par

\dialogue{摘要课程元素}{AI 使用限制、公平正义、技术伦理、神话叙事、叙事权、长时段能力退化、行动哲学。}

\dialogue{人物}{Ta（助教）、s1（组长）、s2（组员）、s3（组员）}

\dialogue{演员}{张灿城 饰 旁白、庞淦匀 饰 Ta、王奕然 饰 s1、胡宸睿 饰 s2、章文浩 饰 s3}

\scene{Scene-0}{庆功宴，实验室内}

\stage{【实验室内。桌上有奶茶、炸鸡、几盒没拆封的水果。投影上显示“课程大作业提交成功”。空气中弥漫着炸鸡味、胜利味，以及一点点学术不端的味道。】}

\dialogue{s1}{我 review 了一下我们的代码，感觉没什么问题，已经提交上去了。虽然规定不让用 AI agent，但是大家这次作业完成得还是不错，大家辛苦了。}

\stage{【s1 说“辛苦了”的时候，s2 正在把 Claude Code 的窗口最小化。动作熟练得像擦除犯罪现场。】}

\dialogue{s2}{不让用 Claude Code 太抽象了。美国有些地方的大学已经大面积开放 AI 使用了，你老中还在停步不前。}

\dialogue{s3}{我没用过，所以真有传得那么邪乎吗？听说用 Claude Code 这种 agent 写代码非常容易上瘾？形成依赖之后就再也戒不掉了？}

\dialogue{s2}{其实用国产的 Kimi、Mimo 还行，成瘾性不强，偶尔写作业用一用，助教也看不出来。}

\dialogue{s3}{你这个说法怎么像公共卫生宣传片里的台词，“我一开始只是偶尔试试”？}

\dialogue{s2}{我觉得不应该污名化技术。技术是中性的，主要看人怎么用。}

\dialogue{s3}{刀也是中性的，关键是你拿它切菜还是切自己的手。}

\dialogue{s2}{哎，外国货才是真爽。Claude Code 的 API，我天天用，感觉助教也看不出来。}

\stage{【s2 狡黠笑。笑容里有 80\% 的侥幸、15\% 的幻觉、5\% 的剩余 token。】}

\dialogue{s2}{就是国内有的 API 店家喜欢掺一点其他模型的 API，纯度不够。想用纯度高的 token 还得是去美国买 Claude Max。最近社区里不是很火的那个 BlueCoder 也很棒，美国的一个计算机系老师，叫什么 Water White，带着他的一个学生 Jessy Pinkman 做出来了一个很棒的 agent，叫 BlueCoder。HumanEval 上 pass@1 直接干到 80\%，不是 pass@10，是 pass@1，简直一发入魂，不要太正。}

\dialogue{s3}{你确定你说的是编程工具，不是drugs？}

\dialogue{s2}{你别说，还真有点那种感觉。}

\dialogue{s1}{逆天，你还是收着点吧。再用多一点，你写一个归并排序估计都费劲了。最近助教在严抓，我要再看一下你那部分的代码。}

\dialogue{s2}{怕啥，绝对查不出来。查出来也无所谓啊，助教不是你实验室的学长吗？动用一下你的人脉啊组长！}

\stage{【伏笔。s1 笑骂。笑容里有一种“我认识人”的自信，也有一种“我可能认识错人”的不祥。】}

\dialogue{s3}{课上不是讲过 Sub Rosa 吗？在玫瑰之下说的话要保密。我们现在是不是也算 under the rose？}

\dialogue{s2}{更准确地说，我们是 under the token。}

\dialogue{s1}{别贫了，我看看提交记录。}

\stage{【s1 打开 Git 记录。屏幕上出现一个陌生名字：TaMud。】}

\dialogue{s1}{诶，这个 TaMud 是谁？}

\dialogue{Ta（消息）}{带着你的组员，拿上电脑，来我的办公室。}

\stage{【众人面色铁青。】}

\scene{Scene-1}{拷问 s2}

\dialogue{Ta}{宸睿带着电脑先进来吧。}

\dialogue{Ta}{来讲一下，这次大作业你负责哪部分？}

\dialogue{s2}{负责后端的数据处理，还有路径搜索那一块。}

\dialogue{Ta}{你的代码看上去质量很高啊？是你自己写的吗？注释写这么全，还都是English？}

\dialogue{s2}{都……都是自己写的，助教，我……我平时比较International。}

\dialogue{Ta}{好。我们先不讨论风格。先说一下你用的什么数据结构，用什么存的，邻接表还是邻接矩阵？你路径搜索用的什么算法？Dijkstra 还是 BFS？优先队列在哪里？复杂度是多少？？}

\dialogue{s2}{用图存的。。。。}

\stage{【s2 慌张沉默。】}

\dialogue{Ta}{你应该知道事实和观点是有区别的。事实上你的代码确实能跑通，但是“你自己会写代码”这只是你的观点。}

\stage{【s2 慌张沉默，随后逐渐愤怒。】}

\dialogue{Ta}{你知不知道根据课程大纲第 26 条，个人代码 AI 覆盖率只要超过 10\% ，我就可以直接让你挂科。你这都有 99\% 了吧，挂你 100 次都不多。}

\dialogue{s2}{我是用了，但是别的课程早就可以用了啊！凭什么就这门课用不了？接纳 AI 就是大势所驱！就这门课搞什么特色课程，要什么公平公正，大家都用不都是公平的吗！}

\dialogue{Ta}{大家都用，就叫公平了吗？}

\dialogue{s2}{不然呢？现在外面都什么环境了？美国那边有的课都开始教学生怎么用 agent 写项目了。人家讲效率，讲工程化，讲 AI-native。我们这边还在问“你是不是自己写的”。这不就是拿旧时代的尺子，量新时代的学生吗？}

\dialogue{Ta}{所以你觉得，会用 Claude Code，就是新时代学生？}

\dialogue{s2}{至少比手搓强。}

\dialogue{Ta}{不知道你听没听说过罗尔斯的“无知之幕”。具体来说就是假设你现在不知道自己会不会用 Claude，不知道自己有没有足够的钱买 API，不知道自己有没有VPN，不知道自己会不会被 AI 写出来的代码欺骗。你还愿意制定“大家随便用，谁跑通谁赢”的规则吗？你还觉得这是公平的吗？}

\dialogue{s2}{可是大家都可以去学啊。}

\dialogue{Ta}{那这么说每个人都能靠努力成为亿万富翁。}

\dialogue{s2}{但 AI 就是工具啊。普罗米修斯盗火也没人说他作弊。}

\dialogue{Ta}{你又扯到神话了。我告诉你，普罗米修斯盗火之后，宙斯为了惩罚他创造了潘多拉，潘多拉的魔盒被打开后放出了各种灾难。技术当然是火，但是这团火可能会烧到你的绩点。问题不是能不能点火，问题是谁点的、在哪里点的、能不能灭火、烧完以后带来了什么进步。}

\dialogue{s2}{你这是技术保守主义。}

\dialogue{Ta}{年轻人，这嘴里主义还挺多。你说的不太准确，我告诉你这应该叫科林格里奇困境。就是技术刚出现的时候，你看不清后果；等你看清了，它已经嵌入你的作业、简历、绩点和自尊心里了。你现在不单单是用 AI 写代码这么简单，你是在让AI逐渐削减你的能力，消磨你的竞争力。}

\dialogue{s2}{我承认我用了。但我也是为了效率。额，助教，那我这个算挂吗？}

\dialogue{Ta}{直接给你挂了效率就很高。哎，你去把奕然叫进来}

\scene{Scene-2}{拷问s1}

\stage{【s1 进门。Ta 看着他，不说话。沉默长到像一段没跑完的单元测试。】}

\dialogue{Ta}{我记得你两年前还是学习委员，每天废寝忘食写代码。一年前还会为了一个 lab 熬通宵，还拿了国奖。三个月前还和我一起兢兢业业地给同学们编写测试题目，你现在状态怎么变成这样了？}

\dialogue{s1}{不是，学长，我真的太想进步了。}

\dialogue{Ta}{你看下你这手臂都练成什么样子了。拉下来看看。这肌肉这么大你说你是学计算机的，谁信？从阿兰图灵开始，哪一个搞计算机的是你这个样子！你哪里来这么多时间健身！你自己应该很清楚！}

\dialogue{s1}{学长，健身也是一种自律。}

\dialogue{Ta}{我看你对自己的身体确实挺自律，但是对组员还挺宽容的。}

\dialogue{s1}{我真不是故意放任他们用 AI。我只是觉得，deadline 太紧了，大家压力都很大。}

\dialogue{Ta}{所以你选择了睁一只眼闭一只眼。}

\dialogue{s1}{当组长也很难啊。我要协调大家，还要保证能按时提交。}

\dialogue{Ta}{协调不是掩盖。我看你们这个小组最成功的不是项目管理，是营销。你把“有人用 AI 写了大部分代码”包装成了“大家都很辛苦”。}

\dialogue{s1}{可是每个人确实都参与了。}

\dialogue{Ta}{参与就等于有贡献、有收获吗。那旁观火灾的人跟消防员不就是同一种人。}

\dialogue{s1}{那能不能放过文浩，他负责写报告，没用AI，跟Sam Altman一样闭源的。}

\dialogue{Ta}{（环顾四周）这样，我们在这说了这么多，影响其他老师办公了，先去趟实验室。}

\scene{Scene-3}{TaMud 是谁}

\stage{【三个人一起回到实验室。Ta 把投影打开。屏幕上是 TaMud 的账号页面。头像是一滩泥。】}

\dialogue{s1}{学长，这个 TaMud 到底是谁，为什么是它提交的代码？}

\dialogue{s2}{是啊，谁把TaMud放我电脑里了？}

\dialogue{Ta}{你承认不是你自己提交的了？}

\dialogue{Ta}{Ta Make U Do It。你们老是说作业太难，所以你们不得不用AI；说是时代在发展，所以你们不得不用AI；说是别人都在用，所以你们也用。最后所有人都说“不是我想做，是环境迫使我做”。你这纯粹就是 TaMud，一滩混着借口、焦虑和 token 的Shit。}

\dialogue{s1}{所以账号是你建的？}

\dialogue{Ta}{是我偷偷写在作业文档里的，如果你直接让agent 读文档，写作业，就会触发用这个账户提交代码。}

\dialogue{s2}{助教，你这不是钓鱼执法吗？}

\dialogue{Ta}{鱼自己跳进锅里，还问锅为什么这么热。}

\dialogue{s3}{那我们是不是已经没希望了？}

\dialogue{Ta}{你们如果不是希望助教心软，而是希望能重新把独立思考，独立创新，综合决策的能力找回来，你们的未来就还有希望。我们之前不是提到了潘多拉魔盒吗，这个潘多拉魔盒最后不还是留了希望在里面吗？好，就先说这么多，今晚还有法国踢塞内加尔，别耽误我看世界杯。}
